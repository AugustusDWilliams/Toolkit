
# GitFlow

```bash
git init
touch .gitignore
git add -A
git commit -m "Initial commit."
git push origin master
git checkout master
git checkout -b develop
```

```bash
git init
touch .gitignore
git add -A
git commit -m "Initial commit."
git push origin master
git branch develop
git push -u origin develop
git checkout develop
git checkout -b feature_branch
git checkout develop
git merge feature branch
```
